1.Download the project file from the repository.
2.Extract the downloaded file to a desired location on the computer.
3.Extract the individual zip files of videos,vscode and images
4.Open Visual Studio Code (VS Code) on the computer.
5.Click on "Open Folder" in VS Code and navigate to the extracted project folder.
6.Select the project folder and click "Open".
7. Click Go live on vs code to run the project.
